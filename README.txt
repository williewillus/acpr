Requirements
============
- Boost Filesystem and System libraries, version 1.58 or higher

Building
========
- Create build/ directory in root
- Within build/, execute "cmake .."
- Within build/, execute "make"
- The binary will be output in "build/acpr"

Cleaning
========
- Simply delete build/
